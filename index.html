Simulador de División de Cuenta

Descripción del Simulador

Esta aplicación web interactiva simula el proceso de dividir la cuenta de un restaurante entre varios comensales. El simulador permite ingresar el monto total de la cuenta, el número de personas que van a pagar, sus nombres, y automáticamente calcula un 10% de propina distribuyéndola equitativamente entre todos los participantes.

La aplicación utiliza funciones JavaScript para capturar datos de entrada mediante cuadros de diálogo interactivos, procesa la información aplicando cálculos matemáticos para la división equitativa, y presenta los resultados de manera clara y organizada a través de la consola del navegador y alertas informativas.

Instrucciones

Abre la consola del navegador (F12) y haz clic en el botón "Iniciar Simulador" para comenzar el proceso de división de cuenta.
